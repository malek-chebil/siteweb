## ğŸš¨ Common Issues & Solutions

### Issue 1: Cannot Connect to VPS
**Solution**:
- Check firewall rules (allow SSH port)
- Check SSH key permissions
- Verify VPS IP address
- Check VPN connection

### Issue 2: Domain Not Resolving
**Solution**:
- Check DNS records in Cloudflare
- Verify nameservers in Njalla
- Wait for DNS propagation (5-30 minutes)
- Check Cloudflare proxy status (should be "Proxied")

### Issue 3: SSL Certificate Not Working
**Solution**:
- Check Nginx configuration
- Verify domain points to server
- Run Certbot: `sudo certbot --nginx -d yourdomain.com`
- Check SSL certificate: `sudo certbot certificates`

### Issue 4: Application Not Loading
**Solution**:
- Check backend service: `sudo systemctl status yourclassifieds-backend`
- Check Nginx: `sudo systemctl status nginx`
- Check logs: `tail -f /var/www/yourclassifieds/backend/logs/app.log`
- Verify environment variables

### Issue 5: Database Connection Failed
**Solution**:
- Check PostgreSQL status: `sudo systemctl status postgresql`
- Verify database credentials in .env
- Check PostgreSQL logs: `sudo tail -f /var/log/postgresql/postgresql-14-main.log`
- Test connection: `psql -U yourdbuser -d yourdbname`
